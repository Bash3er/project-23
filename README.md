# AttendIQ — Employee Attendance & Leave Management System

An Angular application for managing employee attendance and leave requests.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the mock backend (in one terminal):
   ```bash
   npm run json-server
   ```

3. Start the app (in another terminal):
   ```bash
   ng serve -o
   ```

The app will open automatically at `http://localhost:4200`

## Features

- Dashboard with attendance statistics
- Employee listing and profile details
- Daily attendance tracking
- Leave request and approval system
- Role-based access (Admin, HR, Employee)

## Tech Stack

- Angular 17
- Angular Material
- TypeScript
- JSON Server (mock backend)
- SCSS for styling
